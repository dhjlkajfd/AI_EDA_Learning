# Customer Leads / 潜在客户记录

## 使用说明

- 每遇到一个潜在需求就记录一行。
- 重点记录对方问题、我的回复、是否需要跟进。
- 不记录敏感隐私，例如真实姓名、手机号、身份证、学校内部个人信息等。
- 不公开客户联系方式，必要时只用昵称、平台 ID 的模糊描述或群类型代替。
- `Status` 建议使用：`New`、`Replied`、`Following`、`Quoted`、`Closed`、`Rejected`。
