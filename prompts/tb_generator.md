# Testbench Generator Prompt

You are a senior FPGA verification engineer.

Please generate a self-checking Verilog testbench.

## Requirements

- Generate clock
- Generate reset
- Include normal test cases
- Include edge cases
- Automatically check output
- Print PASS or FAIL
- Generate waveform dump if supported

## Output

Please output:

1. File name
2. Complete testbench code
3. How to run simulation
4. Test case explanation
5. Possible missing coverage
